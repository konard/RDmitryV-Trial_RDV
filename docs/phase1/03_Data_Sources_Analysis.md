# Анализ источников данных для маркетинговых исследований

## 1. Категории источников данных

### 1.1 Государственные статистические органы

#### Федеральная служба государственной статистики (Росстат)
- **URL**: https://rosstat.gov.ru/
- **Тип данных**: Официальная статистика по всем отраслям экономики РФ
- **Доступ**: Открытые данные, API
- **Форматы**: CSV, XLS, JSON (через API)
- **Актуальность**: Ежемесячные, квартальные, годовые обновления

**Ключевые разделы данных**:
- Макроэкономические показатели
- Отраслевая статистика
- Региональная статистика
- Демографические данные
- Потребительские цены и инфляция
- Торговля и услуги
- Промышленное производство
- Инвестиции и строительство

**API-доступ**:
- ФГБУ ГМЦ Росстата предоставляет API-сервис для доступа к порталу "Мультистат"
- URL API: https://gmcgks.ru/720
- Формат данных: JSON, CSV
- Ограничения: Требуется регистрация и возможна оплата

**Открытые данные Росстата**:
- URL: https://rosstat.gov.ru/opendata
- Наборы данных доступны для свободной загрузки
- Формат: CSV
- Лицензия: Открытая лицензия

#### Региональные управления статистики
- **Описание**: Статистические управления в субъектах РФ
- **Тип данных**: Детализированная региональная статистика
- **Доступ**: Официальные сайты регионов, запросы
- **Примеры**:
  - Мосгорстат (Москва)
  - Петростат (Санкт-Петербург)
  - Управления статистики областей и краев

**Ключевые данные**:
- Экономические показатели региона
- Демография и занятость
- Розничная торговля и услуги
- Региональные цены

## 2. Оценка источников данных

### 2.1 Критерии оценки источников

#### Достоверность
- **Высокая**: Росстат, министерства, ЦБ РФ, официальные регистры
- **Средняя**: Аналитические агентства, бизнес-информационные системы
- **Низкая**: Форумы, непроверенные блоги

#### Актуальность
- **Высокая**: Новостные агентства, Яндекс.Вордстат, соцсети (real-time)
- **Средняя**: Росстат (квартальные обновления), аналитические отчеты
- **Низкая**: Академические публикации (могут быть устаревшими)

#### Доступность
- **Бесплатные**: Росстат (открытые данные), новостные порталы, Google Trends
- **Условно бесплатные**: Ограниченный доступ к некоторым функциям
- **Платные**: СПАРК, Контур.Фокус, коммерческие исследования

#### Полнота данных
- **Высокая**: Росстат (комплексная статистика), СПАРК (детальная информация о компаниях)
- **Средняя**: Отраслевые ассоциации (фокус на своей отрасли)
- **Низкая**: Отдельные статьи, форумы

### 2.2 Матрица источников по типам исследований

| Тип исследования | Приоритетные источники | Дополнительные источники |
|-----------------|------------------------|--------------------------|
| **Анализ рынка** | Росстат, INFOLine, РБК Исследования | АКРА, отраслевые ассоциации |
| **Региональный анализ** | Росстат, региональные статуправления | Официальные порталы субъектов РФ |
| **Конкурентный анализ** | СПАРК, Контур.Фокус, ЕГРЮЛ | Корпоративные сайты, отзывы |
| **Анализ потребителей** | ВЦИОМ, ФОМ, НАФИ | Социологические исследования, соцсети |
| **Трендовый анализ** | Яндекс.Вордстат, Google Trends, новостные агентства | Научные публикации, международные источники |
| **Ценовой анализ** | Росстат (потребительские цены), парсинг сайтов | Отраслевые обзоры |
| **Анализ рынка труда** | hh.ru Статистика, Росстат | Отраслевые исследования |

## 3. Технические подходы к сбору данных

### 3.1 API-интеграции

#### Доступные API:
1. **Росстат (через ГМЦ)**:
   - Формат: JSON
   - Авторизация: Требуется регистрация
   - Ограничения: Возможны лимиты на запросы

2. **ЦБ РФ (курсы валют)**:
   - URL: https://www.cbr.ru/development/
   - Формат: XML, JSON
   - Доступ: Открытый

3. **Контур.Фокус API**:
   - Данные о компаниях
   - Авторизация: API-ключ
   - Доступ: Платный

4. **hh.ru API**:
   - URL: https://api.hh.ru/
   - Формат: JSON
   - Доступ: Открытый для базовых функций

### 3.2 Веб-скрапинг

#### Законный скрапинг:
- Публичные данные, не защищенные robots.txt
- Соблюдение условий использования сайтов
- Уважительное поведение (rate limiting, user-agent)

#### Инструменты:
- **BeautifulSoup4**: Парсинг HTML
- **Scrapy**: Фреймворк для скрапинга
- **Selenium**: Для динамических сайтов (JavaScript)
- **Playwright**: Автоматизация браузера

#### Объекты для скрапинга:
- Новостные сайты (тренды, упоминания)
- Отраслевые порталы (обзоры, статьи)
- Корпоративные сайты конкурентов (продукты, цены)
- Платформы отзывов (потребительские мнения)

**Важно**: Необходимо соблюдать законодательство РФ и условия использования сайтов.

### 3.3 Загрузка открытых данных

#### Формат CSV/Excel:
- Росстат открытые данные
- Экспорты из статистических порталов
- Обработка с помощью pandas

#### Формат PDF:
- Аналитические отчеты
- Извлечение данных: PyPDF2, pdfplumber
- OCR для сканированных документов: Tesseract

### 3.4 Интеграция с LLM для обработки

#### Использование LLM:
- Извлечение структурированных данных из текста
- Анализ тональности
- Суммаризация больших текстов
- Извлечение ключевых фактов и цифр

#### Модели:
- OpenAI GPT-4
- Anthropic Claude
- Локальные модели (Llama, Mistral) для чувствительных данных

## 4. Система верификации источников

### 4.1 Уровни надежности источников

#### Уровень 1 (Высокая надежность):
- Росстат
- Министерства и ведомства РФ
- ЦБ РФ
- Официальные регистры (ЕГРЮЛ)

**Критерии**: Официальный статус, государственная аккредитация

#### Уровень 2 (Средняя надежность):
- Крупные аналитические агентства (РБК, INFOLine)
- Бизнес-информационные системы (СПАРК, Контур.Фокус)
- Авторитетные социологические центры (ВЦИОМ, ФОМ)
- Отраслевые ассоциации

**Критерии**: Репутация, методологическая прозрачность, коммерческий статус

#### Уровень 3 (Условная надежность):
- Новостные агентства (для трендов)
- Научные публикации (требуется peer-review)
- Корпоративные данные компаний

**Критерии**: Требуется перекрестная проверка

#### Уровень 4 (Низкая надежность):
- Форумы
- Личные блоги
- Непроверенные источники
- Социальные сети

**Критерии**: Использовать только для качественных инсайтов, не для фактов

### 4.2 Проверка актуальности

#### Автоматическая проверка:
- Извлечение даты публикации
- Проверка на устаревание (> 2 лет для большинства данных)
- Сопоставление с более свежими источниками

#### Критичные сроки:
- **Экономическая статистика**: Не старше 1 года
- **Отраслевые тренды**: Не старше 2 лет
- **Технологические тренды**: Не старше 6 месяцев
- **Нормативные документы**: Актуальная редакция

### 4.3 Перекрестная верификация

#### Методы:
1. **Сопоставление данных** из разных источников
2. **Проверка согласованности** показателей
3. **Выявление аномалий** и противоречий
4. **Консенсусное значение** при расхождениях

#### Пример:
Если два надежных источника дают разные оценки объема рынка, система должна:
- Отметить расхождение
- Указать диапазон значений
- Предоставить обоснование каждой оценки
- При возможности вычислить среднее или медианное значение

### 4.4 Маркировка данных

Каждый факт в отчете должен содержать метаданные:
```json
{
  "fact": "Объем рынка X составляет Y млрд руб.",
  "source": "Росстат",
  "url": "https://rosstat.gov.ru/...",
  "date_published": "2025-11-15",
  "reliability_level": 1,
  "verified": true,
  "verification_method": "official_source"
}
```
